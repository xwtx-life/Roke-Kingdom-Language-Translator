# 洛克语翻译器 (Roke Kingdom Language Translator)

这是一个为《洛克王国》爱好者打造的高级语言转译工具。它不仅能将普通文字转换为神秘的洛克符文，还提供了一系列沉浸式的交互体验，旨在为社交媒体引流提供高质量的视觉素材。

## 🌟 核心功能 (Core Features)

### 1. 智能双语翻译
*   **中转英映射**：支持直接输入中文，系统会自动通过 API 将其翻译为英文，再转换为对应的洛克语符号。
*   **原生英文转换**：完美支持 26 个英文字母与洛克符文的 1:1 精确映射。
*   **双向解密**：支持将洛克语符号粘贴回输入框，系统会自动识别并还原为英文含义。

### 2. 沉浸式视觉体验
*   **魔法微粒特效**：输入文字时，翻译区域会自动产生不断升腾的紫色魔法粒子。
*   **玻璃拟态 UI**：采用现代化的暗黑魔法风格设计，包含磨砂玻璃面板、动态光圈渲染以及旋转魔法边框。
*   **官方字体集成**：直接引入腾讯官方 CDN 字体文件，确保符文渲染与游戏内完全一致。

### 3. 洛克语字符对照表
*   **学习参考**：页面底部提供了 26 个字母与符文的实时对照表，点击即可查看对应的魔法字符。

## 🎯 项目目的 (Purpose)

1.  **引流神器**：利用神秘的魔法符文激发用户好奇心，为博主提供新颖的互动方式。
2.  **粉丝工具**：为《洛克王国》社区提供一个好用、好看的免费翻译工具，增强粉丝粘性。
3.  **设计展示**：展示现代 Web 技术（Vite, TailwindCSS, Canvas API, 异步翻译 API）在垂直细分领域的创新应用。

## 🛠️ 技术栈 (Tech Stack)

*   **框架**: Vite + TypeScript
*   **样式**: TailwindCSS 4.0
*   **逻辑**: 原生 Canvas API (用于粒子与图片合成)
*   **API**: MyMemory Translation Service
